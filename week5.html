Github

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

Git

Git is a free and open-source version control system distributed with the Git source code. It is designed to handle everything from small to very large projects with speed and efficiency. Git tracks the changes you make to files, so you can revert to a previous version if you make a mistake.
